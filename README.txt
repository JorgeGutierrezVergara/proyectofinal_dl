Para ejecutar el proyecto:
 1. En backend/consultas.js modificar datos de la database por los del usuario
 2. En backend, iniciar una terminal y ejecutar node index.js
 3. En frontend, iniciar una terminar y ejecutar npm run dev

Actualmente el proyecto puede crear usuario, iniciar sesión y mostrar dos versiones
de la barra de búsqueda dependiendo si se ha iniciado sesión o no. También puede
mostrar en la página de inicio los datos extraídos desde la base de datos.
